jQuery UI dial control
======================

This is a dial widget for jQuery UI.

Options
-------

 * `min`  
   Minimum value.  
   Default: -100

 * `max`  
   Maximum value.  
   Default: 100

 * `default`
   Initial value.  
   Default: 25

 * `unitsPerPixel`
   Amount of units to change the value per pixel of mouse movement.  
   Default: 1

 * `numImages`  
   Number of images in the image map.  
   Default: TODO

 * `imageWidth`  
   Width of an individual dial state image.  
   Default: TODO

Events
------

 * `start`
 * `change`
 * `stop`

Methods
-------

 * `destroy`
 * `disable`
 * `enable`
 * `option` (individual)
 * `option` (hash)
 * `widget`
 * `value` (get)
 * `value` (set)
